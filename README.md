# A commons tool lib

## Download from central

```xml
<dependency>
  <groupId>cn.ponfee</groupId>
  <artifactId>commons-core</artifactId>
  <version>1.0</version>
</dependency>
```

## Build from source

```bash
./mvnw clean package -DskipTests -Dcheckstyle.skip -U
```
